프로젝트명: 예약봇(YEYAKBOT) 웹사이트 MVP
목표: 한국 소상공인 대상 “놓친 전화를 예약으로” 서비스 소개 · 3분 예약 폼 · 20분 데모 예약 페이지를
      가볍고 빠른 정적 사이트(HTML/CSS/JS)로 제작. 한글 기본.

## 파일구조
/ index.html             # 랜딩(CTA 2개: 3분 예약, 20분 데모)
/ booking.html           # 예약 폼 임베드(Google Forms 또는 Tally)
/ demo.html              # 데모 캘린더 임베드(Calendly)
/ robots.txt
/ sitemap.xml
/ assets/logo.svg        # 달력+체크 아이콘 로고(단색, 벡터)
/ assets/favicon.ico
/ assets/og.jpg          # 1200x630 Open Graph 이미지(단색 로고+문구)
/ css/style.css
/ js/app.js

## 디자인/카피(필수 반영)
- 헤드라인: “놓친 전화를 예약으로. 광고비 0원, 첫 달 성과형.”
- 서브: “3분 설치 · 24/7 자동응답 · 즉시 견적 · 콜백 예약. 전환 없으면 그달 구독료 0원.”
- 가격 박스: “세팅비 49만(1회) · 월 29만 · 미팅확정 10만/건”
- 상단 우측 CTA 버튼 2개(고정): [3분 예약] → /booking.html , [20분 데모] → /demo.html
- 푸터: © {연도} 예약봇 · YEYAKBOT / robots, sitemap 링크

## 기술 요구
- 프레임워크 금지(React/Vue/Bootstrap X). 순수 HTML5/CSS/JS.
- 모바일 퍼스트, CLS 0에 가깝게, LCP<2.0s(정적 리소스 최소화).
- style.css: 시스템 폰트 스택, 1100px 컨테이너, 카드 3열 그리드(반응형).
- app.js:
  - 쿼리스트링 UTM을 localStorage에 저장, booking.html/demo.html 링크 클릭 시 UTM 유지.
  - 헤더 고정, 스크롤 시 그림자 처리.
- SEO:
  - <title>, <meta name="description"> 한글.
  - <link rel="canonical" href="https://yeyakbot.com/">
  - Open Graph 태그(og:title, og:description, og:image, og:url).
  - JSON-LD(Organization): 이름=예약봇, url=https://yeyakbot.com, sameAs에 카카오채널 placeholder.
- robots.txt: 전체 허용, sitemap 경로 지정.
- sitemap.xml: /, /booking.html, /demo.html 수록.
- 접근성: 대비비율 충분, 버튼 <a>는 aria-label 지정, 키보드 포커스 스타일 제공.
- 에셋 최적화: logo.svg는 stroke 2px, fill none, 파일 2KB 내외. og.jpg는 40KB±.

## 페이지별 세부
index.html:
- Hero(헤드라인/서브/CTA 2개), 장점 3카드, 가격 박스, 설치·보증·카카오 연동 섹션.
- 헤더 로고: assets/logo.svg + “예약봇 YEYAKBOT” 텍스트.

booking.html:
- 상단 설명문 2줄 후 iframe 임베드.
- 기본은 Google Forms 임베드. src는 “REPLACE_FORM_URL” placeholder로 넣고,
  app.js에서 UTM을 자동으로 붙여주도록 작성.

demo.html:
- Calendly 인라인 위젯 임베드. data-url은 “REPLACE_CALENDLY_URL” placeholder.
- 높이 740px, 반응형.
